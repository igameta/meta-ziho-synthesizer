# これは何？
Voicevoxで音声を生成してmp3にエンコードするためのコードです。
[meta-ziho](https://github.com/igameta/meta-ziho)で再生される音声ファイルを生成します。
5分単位の時報に使用する音声ファイルをmp3で出力します。
実行には(voicevox_core)[https://github.com/VOICEVOX/voicevox_core]が必要です。
